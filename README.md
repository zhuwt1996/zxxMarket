# zxxMarket

## 项目简介

仿淘宝的uni-app项目，目前只开发微信小程序端，后续适配app和h5。

主要的技术栈：

uniapp：https://uniapp.dcloud.io/

vue：https://cn.vuejs.org/

vuex：https://vuex.vuejs.org/zh/

第三方UI组件库：

uView：http://www.uviewui.com/components/intro.html
